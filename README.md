# Medical Clinic Console MVP

Консольное приложение для управления записями на прием к врачам. 
Проект создан как демонстрация базовых навыков владения C# и принципов ООП.

### Что реализовано:
- **Модели данных:** Врачи (`Doctor`) и Слоты записи (`Appointment`).
- **Бизнес-логика:** Система проверки доступности слотов, защита от повторного бронирования и обработка несуществующих ID.
- **Архитектура:** Разделение ответственности между сущностями, использование инкапсуляции и коллекций (`List<T>`).

### Стек:
- C# / .NET 10.0
- Console Application
