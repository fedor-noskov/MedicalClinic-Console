# Medical Clinic Console MVP

Консольное приложение для управления записями на приём к врачам.
Проект создан как демонстрация базовых навыков C# и принципов ООП.

### Что реализовано:
- **Модели данных:** `Doctor`, `Patient`, `Appointment` с валидацией через исключения
- **Бизнес-логика:** Бронирование и отмена записей, защита от повторного бронирования, обработка несуществующих ID
- **Интерфейс `IClinic`:** Бизнес-логика абстрагирована через интерфейс — `Program.cs` работает с `IClinic`, а не напрямую с `Clinic`
- **Разделение ответственности:** Сервисный слой только бросает исключения, вся обработка ошибок на стороне вызывающего кода
- **Архитектура:** Разделение на папки `Services/` и `Interfaces/`, инкапсуляция, коллекции `List<T>`

### Стек:
- C# / .NET 10.0
- Console Application
